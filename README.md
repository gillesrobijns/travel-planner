# Travel Planner

Data-driven trip dashboards. Each trip is a `trip.json` (validated against `schema/trip.schema.json`) injected into a shared HTML template at build time, so every published dashboard is a single self-contained file.

## Structure

- `index.html` — the trips hub: lists every trip and links to its dashboard. Generated by `build-index.js`.
- `trips/<slug>/trip.json` — the source data for a trip.
- `trips/<slug>/dashboard.html` — the built dashboard. Self-contained; nothing is fetched at runtime.

## Adding a trip

Add `trips/<slug>/trip.json`, build the dashboard from the template, then run `node build-index.js` to regenerate the hub.
